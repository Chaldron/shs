# [stevenson.space](https://stevenson.space)
Beautiful and practical agenda management tool for SHS students

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
